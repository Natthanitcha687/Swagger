# 66030232-natthanitcha
Natthanitcha Kitichayachot
สาขาครุศาสตร์วิศวกรรม สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง
19 บ้านณภัฑฐ์ ห้อง 611 แขวงลำปลาทิว เขตลาดกระบัง กทม. 10520

ถึงอาจารย์การ์ตูนที่รัก หนูกิมหงส์รู้สึกประทับใจทุกครั้งที่ได้เรียนกับอาจารย์ค่ะ มีความสุขมากๆ ชอบในการเรียนวิชานี้ อาจารย์น่ารัก รู้สึกสนิทสนมกับอาจารย์เป็นพิเศษ เพราะอาจารย์เป็นมิตรค่ะ
